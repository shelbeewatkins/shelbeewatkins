My Minor
Computer Science
I started taking Computer Science classes because my friend is majoring in Computer Science was working
on some code and I was able to point out one missing semicolon that fixed the code he was struggling on.
My interest started with finding the puzzle aspect fun, but it increased when I realized the practical aspects of it.
Recently, my lab has come to a standstill because we cannot afford to test a molecule that inhibits our target protein,
because the molecule is made by Kansas University and they charge too much money for it. So my lab asked me to help
find analagous molecules as a way to work around this problem. I used a pipelining tool to search for molecular similarity.
This opened my eyes to the world of both Bioinformatics and Chemoinformatics, so that is when I decided to
start a minor in Computer Science.
